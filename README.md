# EduMentor: AI Learning Agent for Low-Resource Students

## Overview
EduMentor is a multi-agent AI learning assistant designed to provide personalized tutoring,
adaptive study plans, and automated content processing for students in low-resource settings. 
It uses LLM-powered agents, memory management, and custom tools to help students learn effectively.

## Features
- Personalized study plans and pacing
- Summaries of PDFs and web content
- Step-by-step explanations for any topic
- Auto-generated quizzes and evaluations
- Motivational coaching and progress tracking
- Multi-agent architecture with sessions and memory

## Architecture
The system includes four cooperating agents:
1. **Content Agent** – Summarizes materials and creates structured notes.
2. **Tutor Agent** – Provides explanations and simplified content.
3. **Evaluation Agent** – Generates quizzes and evaluates responses.
4. **Coach Agent** – Provides motivation and tracks learning progress.
